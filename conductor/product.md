# Initial Concept
- **Objetivo:** Construir una web personal y un ecosistema de contenido automatizado (Zero-Budget) para un Ingeniero en Sistemas especializado en GenAI.
- **Filosofía:** Automatización extrema para delegar el trabajo operativo. El sistema debe funcionar de manera autónoma para liberar tiempo real: el objetivo final es que el código se escriba y publique solo mientras el creador puede estar trabajando en el jardín de su casa en Alta Gracia o jugando con sus hijos de 2 y 5 años.

# Product Guide: Ecosistema "Presupuesto Cero"

## Vision
Construir una web personal y un ecosistema de contenido automatizado (Zero-Budget) para un Ingeniero en Sistemas especializado en GenAI. La filosofía central es la automatización extrema para delegar el trabajo operativo, permitiendo que el sistema funcione de manera autónoma.

## Core Values
- **Automatización Extrema:** El objetivo final es que el código se escriba y publique solo, liberando tiempo real para la vida personal.
- **Presupuesto Cero:** Todo es serverless y estático. Nunca se proponen dependencias pagas ni bases de datos externas.
- **Independencia:** Código modular e independiente.

## Target Audience
- Reclutadores y empresas interesadas en GenAI.
- Comunidad técnica interesada en automatización y "Zero-Budget".
- Seguidores del contenido generado de forma autónoma.

## Key Features
- **Web Personal:** Generada con Astro (SSG).
- **Gestión de Contenido:** Markdown/MDX gestionado vía File-System MCP.
- **CI/CD Automático:** Despliegue mediante GitHub Actions / Vercel.
- **Ecosistema Autónomo:** Capacidad de auto-publicación y actualización.

## Constraints
- Sin dependencias pagas.
- Sin bases de datos externas.
- Todo debe ser estático o serverless (dentro de capas gratuitas).
