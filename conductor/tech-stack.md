# Tech Stack: Ecosistema "Presupuesto Cero"

## Frontend
- **Framework:** Astro (SSG) - Ideal por su ligereza y enfoque en contenido estático.
- **Styling:** Tailwind CSS - Para un diseño rápido y responsive.
- **Componentes:** Astro Components.

## Content Management
- **Formatos:** Markdown (.md) y MDX (.mdx) para contenido rico y programable.
- **Storage:** File-System MCP (Model Context Protocol) para gestión local y automatizada.
- **Estructura:** Localizada en `/src/content/`.

## Infrastructure
- **CI/CD:** GitHub Actions para pruebas y despliegue automatizado.
- **Hosting:** Vercel (Plan gratuito) para un despliegue serverless y global.
- **Dominio:** Integración con DNS gestionado.

## Constraints
- **Presupuesto Cero:** Solo herramientas gratuitas o de código abierto.
- **Serverless/Estático:** Sin necesidad de servidores dedicados ni bases de datos de pago.
