# Specification: 01-fundamentos-ui-astro

## Context
Limpiar la instalación por defecto de Astro, configurar Tailwind CSS con la paleta de colores neutros y formas orgánicas, y crear el layout base.

## Goals
- Eliminar el contenido de bienvenida del starter de Astro.
- Configurar Tailwind con la paleta de colores neutros (tierras, verdes suaves, cremas) y formas redondeadas.
- Crear un Layout base modular con Header, Main y Footer.
- Preparar el área de contenido para inyectar MDX.

## Requirements
- Astro components (`Header.astro`, `Footer.astro`, `Layout.astro`).
- Tailwind CSS configuration.
- Responsive design following `product-guidelines.md`.
