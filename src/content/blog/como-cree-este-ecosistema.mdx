# El origen: Filosofía "Presupuesto Cero"

Construir una presencia digital robusta no debería requerir presupuestos elevados ni mantenimiento manual constante. Este ecosistema nace bajo una premisa clara: **Automatización extrema para delegar el trabajo operativo.**

## El Stack Tecnológico

Para lograr un sistema rápido, escalable y, sobre todo, gratuito en su infraestructura, elegí las siguientes herramientas:

- **Frontend:** [Astro](https://astro.build/) (SSG) para un rendimiento impecable y un enfoque en el contenido.
- **Estética:** [Tailwind CSS](https://tailwindcss.com/) v4, configurado con una paleta de colores neutros y formas orgánicas para una sensación humana y relajante.
- **Contenido:** Markdown y MDX, gestionados localmente.
- **Despliegue:** CI/CD automático con GitHub Actions y hosting en Vercel.

## IA como Socio de Desarrollo

La verdadera magia ocurre en la interacción con agentes de IA. No se trata solo de generar código, sino de orquestar un flujo de trabajo donde la IA comprende el contexto del proyecto, sigue guías de estilo estrictas y mantiene un plan de ejecución (`plan.md`) que actúa como fuente de verdad.

> "El objetivo final es que el código se escriba y publique solo mientras yo disfruto del jardín con mi familia."

## Conclusión

Este es solo el comienzo. La arquitectura está lista para expandirse de forma autónoma. En futuros artículos, profundizaré en cómo los agentes MCP (Model Context Protocol) interactúan con mi sistema de archivos para gestionar este blog.
